# BiliTube

BiliTube helps you paste a YouTube link, find the matching upload on Bilibili, and read comments from both platforms side by side.

## Tech stack

- **Next.js 16** with the App Router
- **React** and **TypeScript**
- **Tailwind CSS** for styling
- **Next.js API routes** for the server side
- **YouTube Data API** and **Bilibili APIs** for video and comment data
- **Turso / libSQL** for storing matched videos and admin corrections

In simple terms: it's a full-stack TypeScript web app built with Next.js, styled with Tailwind, and connected to YouTube, Bilibili, and a small database.
